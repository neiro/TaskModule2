# Как начать работать

Ниже я привожу команды для Windows PowerShell, потому что проект я собирал и проверял именно в этой среде. Под Linux команды будут почти такими же, но активация окружения и некоторые вызовы могут немного отличаться.

1. Откройте проект в корневой папке.
2. Создайте и активируйте виртуальное окружение:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

Для Linux активация обычно выглядит так:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

3. Установите зависимости:

```powershell
pip install -r requirements.txt
```

4. Запустите Jupyter Notebook:

```powershell
jupyter notebook
```

5. Откройте `work.ipynb` и выполните ячейки сверху вниз.

6. Файл `.env` уже лежит в корне проекта, и ноутбук использует его автоматически для подключения к базе.

Важно: в этом проекте `.env` намеренно не исключён из Git. Это осознанный компромисс для учебного проекта: здесь я жертвую безопасностью ради удобства ревьювера, чтобы проект запускался сразу без дополнительной ручной настройки.

# О проекте

Это учебный проект по анализу лояльности пользователей Яндекс Афиши. В работе я выгружаю данные из PostgreSQL, подготавливаю их, строю профиль пользователя и исследую, какие признаки связаны с повторными покупками.

# Что сделано

- выгружены и проверены данные по заказам;
- выручка приведена к рублям;
- отфильтрованы верхние выбросы по выручке и аномально активные профили;
- построен пользовательский профиль;
- выполнен EDA по признакам первого заказа, выручке, количеству билетов и временным характеристикам;
- проведён корреляционный анализ `phi_k`.
